Base URL: http://localhost:5000/api
Timeout: 10000ms
Credentials: withCredentials: true (for refresh token cookies)
Request Interceptor:

Checks sessionStorage.getItem('accessToken') first
Falls back to localStorage.getItem('accessToken')
Parses auth-storage for backward compatibility
Adds Authorization: Bearer ${token} header
Response Interceptor:

Handles 401 errors with message "Access token has expired."
Queues failed requests during refresh
Calls /auth/refresh-token endpoint with cookies
Updates sessionStorage with new access token
Retries original request with new token
Clears auth storage and redirects to /login on refresh failure
